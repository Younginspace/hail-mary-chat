# Hail Mary Chat

> 跟《挽救计划》里的外星人 Rocky 打一通跨星际视频电话。

**[rocky.savemoss.com](https://rocky.savemoss.com)**

灵感来自《挽救计划》（Project Hail Mary）。打开网页，你会看到一个太空飞船通讯界面，Rocky 的 3D 全息投影漂浮在屏幕中央。点击通话，Rocky 会用他标志性的方式回应你——"我是高兴！朋友！"

## 体验

- 3D 全息投影风格的 Rocky 形象
- AI 实时对话 —— Rocky 的性格和语气忠于原著
- 文字转语音（TTS），Rocky 会"说"出来
- 太空飞船通讯界面风格

## Tech Stack

React + TypeScript + Three.js + MiniMax AI (对话 + TTS)

## 本地运行

```bash
npm install
npm run dev
```
